# Keyboard Shortcuts

Key Combo | Description
----------|------------ 
`CTL+F5` | Run the program
`CTL+F8` | Run the selection
`CTL+SHIFT+9` | Wrap in	( )
`CTL+[` |	Wrap in	[ ]
`CTL+SHIFT+[` |	Wrap in	{ }
`Ctrl + ‘`	| Wrap in	‘ ‘
`CTL+SHIFT+‘`	| Wrap in	” “
`CTL-SHIFT-E` | Open function builder to insert a new function
`F8` | Check syntax
