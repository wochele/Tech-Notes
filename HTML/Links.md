# Links

#### Basic link

```html
<a href="url">link text</a>
```

#### Link Within a Page

Create the Anchor

```html
<a name="chapter4"></a>
```

Link to the Anchor

```html
<a href="#YourAnchor">blabla</a>
```

Link to the Anchor from a Different Page

```html
<a href="http://www.yahoo.com#YahoosAnchor">blabla</a>
```
